# Feed the Realm

An MMO game where players explore countless worlds created by the community. With our world creator tool, anyone can design and publish their own worlds for others to enjoy.

## Repositories

| Repository | Description |
|---|---|
| [core-service](https://github.com/FeedTheRealm-org/core-service) | Go monolithic backend powering all game services |
| [backoffice-page](https://github.com/FeedTheRealm-org/backoffice-page) | React admin dashboard for platform management |
| [landing-page](https://github.com/FeedTheRealm-org/landing-page) | React public-facing website |
